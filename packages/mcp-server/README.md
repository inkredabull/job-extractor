# @inkredabull/job-extractor-mcp-server

Model Context Protocol server for secure CV data access.

## Overview

MCP server that provides a secure, local-only interface for accessing CV information following the Model Context Protocol standard.

## Installation

```bash
npm install
```

## Usage

```bash
# Start server (pattern matching)
npm run start

# Start with LLM support
npm run start:llm
```
