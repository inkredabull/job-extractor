# @inkredabull/career-catalyst-extension

Chrome extension for job application assistance and CV-aware responses.

## Overview

Browser extension that provides seamless integration with job application forms, offering CV-aware responses and automated assistance.

## Installation

1. Open Chrome and navigate to `chrome://extensions/`
2. Enable "Developer mode"
3. Click "Load unpacked"
4. Select the `extension` directory
