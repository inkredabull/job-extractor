# @inkredabull/job-extractor-unified-server

Unified server combining CLI server and MCP server functionality.

## Overview

Single Express-based API server that combines both CLI server and MCP server capabilities.

## Installation

```bash
npm install
```

## Usage

```bash
# Start server (pattern matching)
npm run start

# Start with LLM support
npm run start:llm
```

The server runs on port 3000 by default.
